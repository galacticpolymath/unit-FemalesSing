6.SP.B.4 : Display numerical data in plots on a number line, including dot plots, histograms, and box plots.

Assessment Question:
Describe the similarities and differences between dot plots and a histograms.

☆☆☆☆ Level 4 Response (Exceeds Expectations):
Dot plots and histograms both show count data for a single variable. However, dot plots show how many observations (or measurements) there are for each specific value. Histograms simplify the data by "binning" points into ranges of values to make it easier to see patterns in the data.
